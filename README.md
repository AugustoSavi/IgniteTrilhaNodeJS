<h2 align="center">docker-compose commands</h2>

Rodar aplicação

```bash
docker-compose up
```

Parar aplicação

```bash
docker-compose down
```

Acessar container da aplicação

```bash
docker exec -it rentx /bin/bash
```

Acompanhar logs da aplicação

```bash
docker logs rentx -f
```
